## 备忘录（Memento）

### Intent

在不破坏封装的前提下捕获一个对象的内部状态，并在该对象之外保存这个状态，这样可以在以后将对象恢复到原先保存的状态。

### Class Diagram

- Originator：原始对象。
- Caretaker：负责保存好备忘录。
- Menento：备忘录，存储原始对象的的状态。

备忘录实际上有两个接口:
- 一个是提供给 Caretaker 的窄接口：它只能将备忘录传递给其它对象。
- 一个是提供给 Originator 的宽接口，允许它访问到先前状态所需的所有数据。理想情况是只允许 Originator 访问本备忘录的内部状态。
